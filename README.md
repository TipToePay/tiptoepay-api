# tiptoepay-api

### Prerequisites

* Install python 3.x

* Install Pip

* Install Pipenv https://github.com/pypa/pipenv

```
pip install pipenv
```

### Installing

* Specify which version of Python virtualenv should use
```
pipenv --python=/Path/To/python3
```

* Install dependencies
```
pipenv install
```

* Start the shell

```
pipenv shell
```
